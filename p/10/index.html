云朵乐园 新津紫霞山一日游

时间： 2024-01-14   |   分类： 生活   旅游   life     |   阅读： 255 字 ~1分钟

出发前准备

  • 目的地天气，行程规划

交通工具

  • 距离地铁都比较远，最好自驾，两个目的地之间的车程大约45分钟

具体行程

  1. 上午开车到云朵乐园，陪伴小孩玩耍 1.5个小时左右（有自动售卖机）
  2. 午餐在 冯四孃跷脚牛肉（麓湖店）附近处理（附近还有其它的店铺）
  3. 车上午休后，驾车到老君山（45分钟），停车费10元
  4. 开始爬山（比较轻松，单程1个小时）刚入门处有商店，山顶没怎么看到卖水的
  5. 下山
  6. 在新津安排晚餐或者回家就餐

特别提示

  • 云朵乐园旁边不远（1.3KM）有个大型的，有很多游玩设施的公园（红石公园），如果时间足够，可j将云朵乐园改为红石公园。

紫霞山-休息游玩

时间： 2024-01-07   |   分类： 生活   旅游     |   阅读： 283 字 ~1分钟

出发前准备

  • 目的地天气。
  • 水和零食-休闲，小众景点，需要自备水和零食

交通工具

  • 自驾游，导航点：紫霞山，将车顺序停在阳光大道即可（无特定的停车场）
  • 地铁：最近的地铁口10KM，酌情考虑。

攻略

  • 整个爬山有两种选择，第一种是小环线（适合新手和有老人小孩的），预计时间1个半小时（全程）
  • 第二种是爬到山顶，预计全程2个小时？（全程）
  • 下面均是针对小环线的情况说明
  • 从右边进入（从阳光大道看着山的方向），刚开始有个 WC,后续没看到。
  • 刚进入（同上）有水售卖，后续没有。
  • 全程时不时有座椅可供休息
  • 全程道路均有硬化，攀爬难度不高。
  • 下山后，离吃饭的聚集地有一段路程3KM？最好安排吃饭时间。

购房计划

整体规划（时间点）

小孩入学那年前，现在了解的政策是当年4.30号之前需要小孩的户口落在房产上，迁户口大约需要2周

主要目标

  1. 学校可以
  2. 房子可以
  3. 配套可以
  4. 环境可以

预算

  • 不超过220W

备选小区

花样年美年广场-美岸

  1. 学区房，高密度（周围没有环绕，给人感觉还还），
  2. 远离老破小的市中心，逛街不方便，但新开发的区域，周围环境还不错
  3. 房价有些超预期，可以再等等
  4. 河边新修的路可能要通车了，噪声会比较大
  5. 有些一面之缘的人，人际关系好
  6. 有物业的工作，无需再次找工作

二十一世纪花园

  1. 离公司近
  2. 3，4期不靠近高架
  3. 没有电梯
  4. 总价不高（210W内）

棕南公寓

  1. 特点 学区房， 低密度 ，位置不错
  2. 缺点 小区偏旧

万科城市花园

thrift 之 go 入门 Getting Started with Thrift Go

时间： 2023-12-08   |   分类： IT   rpc   thrift     |   阅读： 286 字 ~1分钟

下载、安装 thrift 编译器

  1. thrift 编译器的下载连接 https://thrift.apache.org/downloa 这里注意下版本
  2. 将下载的编译器放到系统的可执行目录下，并去掉版本信息， eg: thrift-0.19.0.exe 改为 thrift.exe

go 安装 thrift 包

go get github.com/apache/thrift@0.19.0
#这里的版本要和上面的对应

新建 tmp 目录

新建一个临时目录 tmp
在新建的 tmp 目录下新建一个存放 thrift 文件的 thrift 目录
将上述 thrift 包的实例文件(tutorial)拷贝到 thrift 目录下
我这里的 thrift 安装包的路径是 C:\Users\shanks\go\pkg\mod\github.com\apache\thrift@v0.19.0 
将 tutorial 中的 shared.thrift 和 tutorial.thrift 文件 拷贝到 tmp/thrift中

生成对应语言的协议文件

在上述新建的 tmp 目录下执行 
thrift -r --gen go thrift/tutorial.thrift
生产 go 语言版本的 thrift协议代码

拷贝实例代码

将上述安装目录下的 tutorial\go\src 中的代码（包括上层的 server.crt 和 server.key ）拷贝到 tmp 目录下

编译 go 代码

go mod init example.com/m 有 go.mod 可不执行
go build main.go
编译报错处理即可
